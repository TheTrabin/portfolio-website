First Unity Game

While learning how to use Unity, this was one of the first projects.

The game utilizes Keyboard and Mouse in order to play, so sadly Tablet and Phone players can't join in.

The point of the game is to run up and collect 'coins' and rush back to the start, or gate.

The game is hosted via applet compatible with HTML5.
